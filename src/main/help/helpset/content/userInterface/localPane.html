Local Pane

The local pane displays files and directories on the local file system. The local pane contains the following components:

Root Directories
The drop-down box on the top left corner lists all root directories available on the local system. Selecting a root directory from the list changes the working directory to the selected root directory.

On most systems, there will be just one root directory. For e.g. Desktop is the root directory on Windows and "/" is the root directory on all UNIX type systems.

Working Directory
The drop-down box next to the root directories displays the current working directory. This drop-down box displays a tree like structure starting at the root directory and ending at the working directory. Selecting an entry from the drop-down box, changes the working directory to the selected directory.

Up Button
To traverse one level up in the directory tree, click on the  button.

Directory List Table
The directory list table displays the contents of the working directory. This table contains four columns to display the name, type, size and modification date of the files.

File Count and Size
Just below the directory list table is a status bar to display the total number of objects in the working directory, size of all objects, number of selected objects and the size of selected objects.
