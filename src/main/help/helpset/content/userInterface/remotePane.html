Remote Pane

The remote pane displays the files and directories on the remote system (FTP site). The remote pane contains the following components:

Working Directory
The drop-down box in the top-left corner displays the working directory on the remote system. In addition, this drop-down box maintains a history of previously visited directories. Selecting a directory from this list changes the working directory to the selected directory.

Up Button
To display the contents of the parent directory of the current directory, click on the   button located in the top right corner.

Directory List Table
The directory list table displays the contents of the working directory. This table contains five columns to display the name, type, size, modification date and the attributes of the files.

File Count and Size
Just below the directory list table is a status bar to display the total number of objects in the working directory, size of all objects, number of selected objects and the size of selected objects.
