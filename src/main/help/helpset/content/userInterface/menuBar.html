Menu Bar

The menu bar is a collection several menus. A menu contains one or more menu items. Each menu item is used to perform a specific task. JFTP's menu bar contains the following six menus:

  1. FTP Menu
  2. Local File Menu
  3. Remote File Menu
  4. Transfer Mode Menu
  5. Tools Menu
  6. Help Menu

FTP Menu

The FTP menu contains the following menu items:

Menu Item Description
New Session Opens a new FTP session tab.
Close Session Closes the selected FTP session tab.
Connect... Opens the Connect dialog.
Disconnect Disconnects the selected FTP session.
Reconnect Reconnects to the previously connected FTP site.
Download Copies files and/or directories from the FTP site to the local system.
Download As... Opens the Download As dialog.
Download and Unzip... Opens the Download and Unzip dialog.
Upload Copies selected files and/or directories from the local system to the FTP site.
Upload As... Opens the Upload As dialog.
Zip and Upload... Opens the Zip and Upload dialog.
Abort Aborts the current transfer, if any
Exit Terminates all active FTP sessions and exits the application

On Mac OS X, the Quit menu item under the JFTP menu is equivalent to the Exit menu item.

Local File Menu

The Local File menu contains the following menu items:

Menu Item Description
Open Opens the selected local file or directory using the associated application.
Edit Opens the selected local file for editing using the associated application.
Print Prints the selected local file using the associated application.
E-Mail Composes a new e-mail message with the selected local file attached to the message.
New -> File... Opens the New Local File dialog.
New -> Directory... Opens the New Local Directory dialog.
Change Directory... Opens the Change Local Directory dialog.
Rename... Opens the Rename Local File or Directory dialog.
Delete... Deletes selected files and directories from your PC, permanently.
Apply Filter... Opens the Filter Local Files dialog.
Clear Filter Clears the current filter, if any.
Refresh Refreshes the contents of local working directory. The root directories will also be refreshed.
Properties Displays the properties of selected local file or directory.
Select All Selects all objects in the local pane.
Invert Selection Inverts the object selection in the local pane.

Remote File Menu

The Remote File menu contains the following menu items:

Menu Item Description
Open Opens the selected remote file or directory using the associated application.
Edit Opens the selected remote file for editing using the associated application .
Print Prints the selected remote file using the associated application.
E-Mail Composes a new e-mail message with the selected remote file attached to the message.
New -> File... Opens the New Remote File dialog.
New -> Directory... Opens the New Remote Directory dialog.
Change Directory... Opens the Change Remote Directory dialog.
Rename... Opens the Rename Remote File or Directory dialog.
Delete... Deletes selected files and directories from the FTP site.
Execute Command -> Help Sends HELP command to the FTP site and displays the response in the message log.
Execute Command -> Status Sends STAT command to the FTP site and displays the response in the message log.
Execute Command -> System Info Sends SYST command to the FTP site and displays the response in the message log.
Execute Command -> No Operation Sends NOOP command to the FTP site and displays the response in the message log.
Execute Command -> Other... Opens the Execute Command(s) dialog.
Apply Filter... Opens the Filter Remote Files dialog.
Clear Filter Clears the current filter, if any.
Refresh Refreshes the contents of remote working directory.
Properties Displays the properties of selected remote file or directory.
Select All Selects all objects in the remote pane.
Invert Selection Inverts the object selection in the remote pane.

Transfer Mode Menu

The Transfer Mode menu contains the following menu items:

Menu Item Description
Auto Detect Sets the transfer mode to auto.
ASCII Sets the transfer mode to ASCII
Binary/Image Sets the transfer mode to binary (also called image)

Tools Menu

The Tools menu contains the following menu items:

Menu Item Description
Add to Favorites... Opens the Add Favorite FTP Site dialog
Manage Favorites... Opens the Manage Favorites dialog
Manage Certificates... Opens the Manage Certificates dialog
Preferences Opens the Preferences dialog.

On Mac OS X, the Preferences menu item is not available under Tools menu, instead, it is available under the JFTP menu.

Help Menu

The Help menu contains the following menu items:

Menu Item Description
Contents and Index Displays help.
Request License... Opens Request a Commercial License dialog
Install License... Opens the Install License dialog
Tips Displays Tips
FAQ Displays FAQ (Frequently Asked Questions).
Support Displays support information.
Send Feedback Provides information on how to send feedback about JFTP.
About Displays product's information.

On Mac OS X, the About menu item does not exist under Help menu, instead, it is under the JFTP menu.
