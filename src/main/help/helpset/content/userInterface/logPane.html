Log Pane

The log pane displays the conversation between JFTP and the FTP site you are connected to. The log pane displays messages in various colors for various types of messages. The table below explains the meanings of these colors:

Message Color Message Type Message Description
Blue FTP commands FTP commands sent by JFTP requesting the FTP server to take the specified action. Please note that all commands are prefixed with "<COMMAND>"
Green FTP replies All replies received from the FTP server. Please note that all positive replies are prefixed with "<REPLY>".
Red Errors All negative replies and other errors, such as, network failures, occurred during the FTP process.
Orange Status messages Status messages. All status messages will be prefixed with "<STATUS>".
