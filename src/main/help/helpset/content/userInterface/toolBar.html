Tool Bar

The tool bar provides quick access to some of the most frequently used tasks in JFTP. Below is the list of available tasks from JFTP's tool bar.

Button Description
Opens a new FTP session.
Displays the Connect dialog.
Disconnects the selected FTP session.
Reconnects to the previously connected FTP site.
Downloads files and/or directories from the FTP site to the local system.
Uploads files and/or directories from the local system to the FTP site.
Aborts the current file transfer, if any.
Displays Create New Local File dialog.
Displays Create New Local Directory dialog.
Displays Change Local Working Directory dialog.
Displays Rename Local File or Directory dialog.
Displays a dialog to confirm the deletion of selected local file(s).
Displays Create New Remote File dialog.
Displays Create New Remote Directory dialog.
Displays Change Remote Working Directory dialog.
Displays Rename Remote File or Directory dialog.
Displays a dialog to confirm the deletion of selected remote file(s).
Displays Manage Favorites dialog.
Displays Manage Certificates dialog.
Displays Help.
