Status Bar

Status bar is used to let the user know what JFTP is doing currently. For e.g. If JFTP is currently downloading or uploading a file etc. Please note that every session will have its own status bar.

The status bar is divided in to five different parts to display the following information:

  1. Status message
  2. Percentage of bytes transferred or processed
  3. Elapsed time since a process has begun
  4. Average number of bytes transferred or processed per second
  5. The type of session – secured or unsecured. A lock symbol in the right corner indicates that the connection is secured.
