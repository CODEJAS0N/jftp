Viewing File/Directory Properties

To view the properties of a file or a directory, select the file or directory that you are interested in and, do any one of the following:

  • Select Local File -> Properties menu item from the menu bar
  • Press Ctrl + Shift + I (Command + Shift + I on Mac OS X) on your key board
  • Right click on the file or folder for which you want to view the properties and select Properties from the context menu

When you do any of the above actions, the Properties dialog box is displayed containing the following information:

Name
Displays name of the file or directory of which you are viewing the properties.

Full Path
Displays full path of this file or directory.

Type
Displays type of this file.

Size
Displays size of this file or directory. If the selected object is a directory, it will be processed recursively to display the total size of all files and sub directories underneath it.

If you are viewing properties of a directory and, have a filter setup, the size will only be calculated for files that match your filter criteria.

Contains
Displays total number of files and sub directories in the selected directory.

This field is displayed if and only if the selected object is a directory. If you have a filter setup, the count will include only those files that match your filter criteria.

Last Modified
Displays the date and time this file was last modified.

Attributes
Displays attributes of the selected file or directory. These attributes include, whether or not this file is readable, writable or hidden.

Close
Clicking on this button closes the properties dialog.

Help
Clicking on this button displays this help topic.
