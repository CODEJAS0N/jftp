Browsing Files and Directories

To display the contents of a root directory (or drive), select a root directory from the Root Directories drop-down box located in the top left corner of the local pane. This will change the working directory to the selected root directory and the contents of the root directory are displayed in the directory list table.

Most computers will have just one root directory, for e.g. Desktop on Windows and "/" on Mac OS X and other UNIX type systems.

To display the contents of a directory that is listed in the directory list table, just double click on that directory or right click on the directory and, choose Open from the context menu.

To traverse one level up in the directory tree, click on the Up button located in the top-right corner of the local pane.

If you have a filter set up, only files that match your filter criteria are displayed in the directory list table. All newly opened sessions will have a default filter set to exclude hidden files and folders.
