Deleting Files and Directories

To delete files and/or directories, select the files and/or directories that you would like to delete and do any one of the following:

  • Press the Delete key on your keyboard (This will work if and only if the directory list table has focus)
  • Select the menu item Local File -> Delete from the menu bar
  • Click on the   button in the tool bar
  • Press Ctrl + Shift + Delete (Command + Shift + Delete on Mac OS X) key on your key board
  • Right click on any of the selected files in the local pane and select Delete from the context menu

When you do any of the above actions, a confirmation dialog will be displayed asking you if you really want to delete the selected objects.

If you answer Yes the selected objects will be deleted permanently from the local system. Clicking No would cancel the delete action.

If you have a filter set up, only files matching your filter criteria are deleted. To delete all files and directories make sure you clear the filter.
