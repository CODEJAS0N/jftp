Opening a File or Directory

To open a local file or directory, do any one of the following:

  • Double click on the file or directory that you would like to open
  • Select (highlight) the file or directory that you would like to open and do any one of the following:
    • Select Local File -> Open menu item from the menu bar.
    • Right click (Ctrl + Click on Mac OS X) on the selected file or directory and choose Open from the context menu
    • Press Ctrl + Shift + O (Command + Shift + O on Mac OS X)

If the selected object is a file, the file will be opened using the associated application. If no application is associated with the selected file, an error message will be displayed indicating the same.

If the selected object is a directory, the working directory will be changed to the selected directory and its contents are displayed in the directory list table.

Opening files using the associated applications may not work on some operating systems, in which case, you will see an error message.
