Printing a File

To print a local file, select (highlight) the file that you would like to print and do any one of the following:

  • Select Local File -> print menu item from the menu bar.
  • Right click (Ctrl + Click on Mac OS X) on the selected file or directory and choose print from the context menu
  • Press Ctrl + Shift + P (Command + Shift + P on Mac OS X)

When you do any of the above actions, the file will be opened for printing using the associated application.

Printing files using the associated applications is not supported on some operating systems, in which case, the Print menu item will be grayed out.
