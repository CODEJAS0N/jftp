Refreshing Contents

To refresh the contents of the current local working directory, do any one of the following:

  • Select Local File -> Refresh menu item from the menu bar
  • Press Ctrl + Shift + F5 (Command + Shift + F5 on Mac OS X) on your key board
  • Right click any where in the local pane and choose Refresh from the context menu

Refresh command will also refresh the system's root directories.
