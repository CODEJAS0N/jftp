E-Mailing a File

To E-mail a local file, select (highlight) the file that you would like to E-mail and do any one of the following:

  • Select Local File -> E-Mail menu item from the menu bar.
  • Right click (Ctrl + Click on Mac OS X) on the selected file or directory and choose E-Mail from the context menu
  • Press Ctrl + Shift + M (Command + Shift + M on Mac OS X)

When you do any of the above actions, the file will be opened for E-mailing using the default E-mail application.

E-mailing of files is not supported on some operating systems, in which case, you will see an error message.
