Viewing File/Directory Properties

To view the properties of a file or a directory, select the file or directory in the remote pane and do any one of the following:

  • Select Remote File -> Properties menu item from the menu bar
  • Press Ctrl + I (Command + I on Mac OS X) on your key board
  • Right click on the file or folder for which you want to view the properties and select Properties from the context menu

When you do any of the above actions, the Properties dialog is displayed.

Name
Displays name of the file or directory of which you are viewing the properties.

Site
The FTP site to which this file or directory belongs to.

Full Path
Displays full path of this file or directory on the remote system.

Type
Displays type of this file.

Size
Displays size of this file or directory. If the selected object is a directory, it will be processed recursively to calculate the total size of all files and sub directories underneath it.

If you are viewing properties of a directory and, have a filter set up, the size will only be calculated for files that match your filter criteria.

Contains
This displays the total number of files and sub directories in the selected directory.

This field is displayed if and only if the selected object is a directory. If you have a filter setup, it will be honored when counting the number of files.

Last Modified
Displays the date and time this file was last modified on.

Links
Displays number of symbolic links that point to this file.

Owner
Displays the owner's of this file.

Group
Displays the group name to which the owner of this file belongs to.

Attributes
Displays various permissions the owner, group and public have on this file.

Apply these attributes to all subfolders and files
Check this check box to apply your changes to all subfolders and files of this directory. Unchecking this check box would apply your changes just to this folder.

Please note that this check box will not be available, if the object you have selected is not a directory.

OK
Clicking this button would close the properties dialog and applies your changes (changes you might have made to the attributes).

Cancel
Discards your changes and closes the properties dialog.

Help
Clicking on this button displays this help topic.
