E-Mailing a File

To E-mail a remote file, select (highlight) the file that you would like to E-mail and do any one of the following:

  • Select Remote File -> E-Mail menu item from the menu bar.
  • Right click (Ctrl + Click on Mac OS X) on the selected file or directory and choose E-Mail from the context menu
  • Press Ctrl + M (Command + M on Mac OS X)

When you do any of the above actions, the file will be opened for E-mailing using the default E-mail application.

The remote file will first be downloaded to a temporary location before it can be e-mailed. E-mailing of files is not supported on some operating systems, in which case, you will see an error message.
