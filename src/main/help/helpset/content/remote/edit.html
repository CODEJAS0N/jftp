Editing a File

To edit a remote file, select (highlight) the file that you would like to edit and do any one of the following:

  • Select Remote File -> Edit menu item from the menu bar.
  • Right click (Ctrl + Click on Mac OS X) on the selected file or directory and choose Edit from the context menu
  • Press Ctrl + E (Command + E on Mac OS X)

When you do any of the above actions, the file will be opened for editing using the associated application.

The remote file will first be downloaded to a temporary location before it can be edited. Editing files using the associated applications is not supported on some operating systems, in which case, the Edit menu item will be grayed out.
