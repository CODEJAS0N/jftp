Executing Commands

Using JFTP, you can manually execute FTP commands to perform some advanced tasks. These commands include the standard FTP commands that were defined in FTP protocol specification (For e.g. RFC 959) and also your FTP site specific commands. To execute commands, select Remote File -> Execute Command menu item from the menu bar. Select a command from the available list to execute that command. To execute a command that is not listed in the menu, select the sub-menu item Other... . This will open the Execute Command(s) dialog.

Enter one or more FTP commands below
In this text box, enter one or more FTP commands that you would like to execute. Make sure to enter only one command per line.

Execute
Clicking this button will execute the commands you have entered. The responses to the commands will be displayed in the Message Log.

You can not execute the following commands from the Execute Command(s) dialog.

LIST
NLST
RETR
STOR
STOU
APPE
PASV
PORT
DATA
CWD
CDUP

Please note that, other menu options are provided to execute most or all of the above commands.

Cancel
Clicking this button will discard your input and closes the Execute Command(s) dialog.
