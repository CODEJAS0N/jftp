General

To view or change the general preferences, open the Preferences dialog by selecting Tools -> Preferences ( JFTP -> Preferences on Mac OS X) menu item from the menu bar. Click on the Connection Settings -> General category in the left frame. This will display the existing general preferences.

Email Address
Enter your e-mail address in this field. The e-mail address is used when you want to log on to anonymous FTP sites. The default installation of JFTP sets this to you@yourcompany.com .

Default Local Directory
Enter the directory that you would like to set as the default local working directory. If you specify a valid directory in this field, in every new session you open, the local working directory is set to the specified directory. The default value for this preference is operating system dependent. On Windows, this defaults to My Documents and on UNIX type platforms this defaults to the user's home directory.
