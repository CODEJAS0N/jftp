User Interface

To view/change the user interface settings of JFTP (how the user interface looks), open the preferences dialog by selecting Tools -> Preferences ( JFTP -> Preferences on Mac OS X) menu item from the menu bar and click on the User Interface category in the left frame. This will display the existing user interface preferences.

Look and Feel
The Look and Feel displays all installed look and feels in your JRE (Java Runtime Environment). The Look and Feel displays all installed look and feels in your JRE (Java Runtime Environment). The items in this drop-down box vary depending on your operating system and JRE implementation.

Select a look and feel from the list to change the look and feel.

The default installation of JFTP sets the look and feel to the platform's native look and feel; i.e. On Windows, the look and feel is set to Windows native look and feel. On Mac OS X, it will be set to Aqua look and feel. On all other operating systems, the default is set to Java look and feel.
