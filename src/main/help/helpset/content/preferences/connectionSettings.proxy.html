Proxy

To view or change the settings of your Proxy server, open the Preferences dialog by selecting Tools -> Preferences ( JFTP -> Preferences on Mac OS X) menu item from the menu bar. Click on the Connection Settings -> Proxy category in the left frame. This will display the existing proxy preferences.

Please note that JFTP currently supports SOCKS proxy only. JFTP automatically detects the version of your SOCKS proxy (version 4 or version 5) and works with it accordingly.

Use the following SOCKS proxy
Check/Uncheck this check box to tell JFTP that you want/do not want to use a SOCKS proxy when connecting to FTP sites.

Host Name
Specify the host name or IP address of your SOCKS proxy server. If you are unsure about this parameter, please contact your network administrator.

Port Number
Specify the port number on which your SOCKS proxy server is running on. The default port number is 1080. If your SOCKS server is running on a port other than the default, enter that port number here. If you are not sure about this parameter, please contact your system administrator.

User Name
If your SOCKS proxy requires authentication, enter your user name (login name) here.

Password
If your SOCKS proxy requires authentication, enter your password here.

When you are done, click on the Save button to save all your changes. This would also close the Preferences dialog.
