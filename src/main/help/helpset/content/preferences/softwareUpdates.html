Software Updates

To view/change the software updates preferences, open the preferences dialog by selecting Tools -> Preferences ( JFTP -> Preferences on Mac OS X) menu item from the menu bar and click on the Software Updates category in the left frame. This will display the existing preferences.

Periodically check for software updates
Check this check box to let JFTP check for the availability of newer versions. With this option checked, JFTP checks for new version every time you start JFTP.

Check Now
Clicking on this button, immediately checks for the availability of newer versions.
