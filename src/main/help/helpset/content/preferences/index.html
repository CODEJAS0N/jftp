Preferences

To view or change the global preferences, select the menu item Tools -> Preferences ( JFTP -> Preferences on Mac OS X) from the menu bar. This will open the Preferences dialog.

The Preferences dialog has various categories listed in the left frame. Selecting a category from the list changes the screen in the right frame. To learn more about a specific category, click on any of the topics below.

  • Connection Settings
    • General
    • Advanced
    • Security
    • Proxy
  • Transfer Modes
  • Regional Settings
  • User Interface
  • Software Updates

Save
Clicking this button saves all changes you have made to the preferences and closes the Preferences dialog.

Cancel
Clicking this button discards your changes and closes the Preferences dialog.

Help
Clicking this button displays this help topic.
