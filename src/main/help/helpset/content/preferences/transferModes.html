Transfer Modes

The Transfer Modes preferences allow you to associate a transfer mode to a file type. JFTP uses these settings when you are transferring files in Auto-Detect mode. To change the Transfer Modes preferences, open the Preferences dialog by selecting Tools -> Preferences ( JFTP -> Preferences on Mac OS X) menu item from the menu bar. Select Transfer Modes category in the Preferences dialog. This will display the existing Transfer Modes preferences.

As you can notice from the Preferences dialog, various file types (file extensions) are associated with a specific transfer mode.

To associate a transfer mode to a new file type, scroll to the end of the table and click in the File Type cell and enter the file type (extension of the file) and press the Enter key. For e.g. DOC . Click in the Transfer Mode cell and select the mode in which you would like to transfer the file.

To remove an association, locate and click on the file type, and clear the File Type field and press Enter key.

Default Transfer Mode
Specify the transfer mode to use if JFTP comes across a file type that is not listed in the transfer modes preferences table. It is always better to use Binary as the default transfer type.

When you are done, you may click on the Save button to save the changes or click on another category to change the preferences in that category.
