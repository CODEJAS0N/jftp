Uploading Files

Upload means transferring files and/or directories from your PC to the FTP site. To upload files, follow the steps given below:

  1. If you are not already connected to the target FTP site, connect now
  2. In the remote pane, change the working directory to the one that you would like to transfer files to
  3. In the local pane, select (highlight) the files and/or directories that you would like to transfer
  4. When you are done, to start the file transfer, do one of the following:
    • Drag the selected files from the local pane and drop them in the remote pane
    • Select the menu item FTP -> Upload from the menu bar
    • Click on the  icon in the tool bar
    • In the local pane, right click on any one of the selected files and select Upload from the context menu
    • Press F9 key on your key board
    When you do any one of the above actions, JFTP will start transferring the selected files.

While the transfer is in progress, you can see the status of the transfer in the status bar.
