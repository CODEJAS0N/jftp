Download and Unzip

Yet another feature of JFTP is the ability to unzip compressed files soon after downloading. Follow the steps given below in order to do this:

  1. If you are not already connected to the source FTP site, connect now
  2. In the local pane, change the working directory to the one that you would like to unzip the contents of the zip file
  3. In the remote pane, select (highlight) a single zip file that you would like to download and unzip and, do one of the following:
    • Select the menu item FTP -> Download and Unzip... from the menu bar
    • In the remote pane, right click on the selected zip file and choose Download and Unzip... from the context menu
    • Press Ctrl + Shift + F7 (Command + Shift + F7 on Mac OS X) key on your key board
    When you do any one of the above actions, a new dialog titled Download and Unzip will be displayed.

Unzip Options

Unzip the file to the current local working directory
Selecting this option causes the files to be extracted to the current local working directory.

Unzip the file to the directory specified below:
Select this option if you would like to extract the files to a different directory than the current local working directory. Selecting this option opens up the file dialog using which you can select the directory where you would want the files to be extracted.

Browse...
Clicking this button opens the file dialog using which you can select the directory where you would want the files to be extracted.

Download Options

Download the files to a temporary directory and delete it after unzipping
Use this option if you do not want to keep the zip file that you are downloading on your local system. With this option selected, JFTP downloads the zip file to a temporary location, unzips it to the specified directory and deletes the zip file from the temporary location.

Download the zip file to the current local working directory
Use this option if you would like to download the zip file to the current local working directory.

Download the file to the directory specified below:
Select this option if you would like to download the zip file to a different directory. Selecting this option opens up a file dialog using which you can locate and select the directory where you want to download the zip file.

Browse...
Clicking this button opens up a file dialog using which you can locate and select the directory where you want to download the zip file.

Download
When you are done selecting the download options and unzip options, click on this button to start the download and unzip process.

Cancel
Clicking this button closes the Download and Unzip dialog.

Help
Clicking on this button opens this help topic.

While the process is in progress, you can see the status in the status bar.
