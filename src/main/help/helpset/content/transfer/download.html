Downloading Files

Download means transferring files and/or directories from the FTP site to your PC. To download files and/or directories, follow the steps given below:

  1. If you are not already connected to the source FTP site, connect now
  2. In the local pane, change the working directory to the one that you would like to transfer files to
  3. In the remote pane, select (highlight) the files and/or directories that you would like to transfer
  4. When you are done, to start the file transfer, do one of the following:
    • Drag the selected files from the remote pane and drop them in the local pane.
    • Select the menu item FTP -> Download from the menu bar
    • Click on the  button in the tool bar
    • In the remote pane, right click on any one of the selected files and choose Download from the context menu
    • Press F7 key on your key board
    When you do any one of the above actions, JFTP will start downloading the selected files.

While the transfer is in progress, you can see the status in the status bar.
