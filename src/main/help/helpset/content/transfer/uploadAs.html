Upload As

The Upload As feature lets you upload a single file or a single directory (and its contents) as a different name. To do this, follow the steps given below:

  1. If you are not already connected to the target FTP site, connect now
  2. In the remote pane, change the working directory to the one that you would like to transfer the file or directory to
  3. In the local pane, select (highlight) the file or directory that you would like to transfer and do any one of the following:
    • Select the menu item FTP -> Upload As...from the menu bar
    • In the local pane, right click on the selected file or directory and, select Upload As... from the context menu
    • Press Ctrl + F9 (Command + F9 on Mac OS X) key on your key board
    When you do any one of the above actions, a new dialog titled Upload As will be displayed.

Upload 'xxx' As
In this field, specify a new name for the selected file or directory.

Upload
Click on this button to transfer the file or directory.

Cancel
Clicking on this button closes the Upload As dialog.

Help
Clicking on this button, displays this help topic.

While the transfer is in progress, you can see the status of the transfer in the status bar.
