Download As

The Download As feature lets you download a single file or a single directory (and its contents) as a different name. To do this, follow the steps given below:

  1. If you are not already connected to the source FTP site, connect now
  2. In the local pane, change the working directory to the one that you would like to transfer file or directory to
  3. In the remote pane, select (highlight) a single file or directory that you would like to download and, do one of the following:
    • Select the menu item FTP -> Download As... from the menu bar
    • In the remote pane, right click on the selected file or directory and select Download As... from the context menu
    • Press Ctrl + F7 (Command + F7 on Mac OS X) key on your key board
    When you do any one of the above actions, a new dialog titled Download As will be displayed.

Download 'xxx' As
Specify a new name for the file or directory that you are downloading.

Download
Click on this button to download the file or directory.

Cancel
Click on this button to close the Download As dialog.
