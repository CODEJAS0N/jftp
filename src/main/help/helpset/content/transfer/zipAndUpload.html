Zip and Upload

Using the Zip and Upload feature, you can compress one or more files and/or directories in to a single zip file and then upload the zip file to your FTP site, all in one easy step. To do this -

  1. If you are not already connected to the target FTP site, connect now
  2. In the remote pane, change the working directory to the one that you would like to transfer the zip file to
  3. In the local pane, select (highlight) the files and/or directories that you would like to zip and upload and, do any one of the following:
    • Select the menu item FTP -> Zip and Upload... from the menu bar
    • In the local pane, right click on any of the selected objects and choose Zip and Upload... from the context menu
    • Press Ctrl + Shift + F9 (Command + Shift + F9 on Mac OS X) key on your key board
    When you do any one of the above actions, a new dialog titled Zip and Upload will be displayed.

Zip File Name
Specify a name for the target zip file

Create zip file in a temporary directory and delete it after uploading
Select this option if you do not want to keep the zip file on your local system. JFTP creates the zip file in a temporary directory, uploads it and then deletes the zip file from the temporary location.

Create zip file in a temporary directory and delete it after uploading option is selected by default.

Create zip file in the current working directory
Select this option to create the zip file in the current local working directory. The zip file will stay in the working directory even after the file is transferred.

Create zip file in the directory specified below:
Use this option if you would like to create the zip file in a different location. Selecting this option would bring up a file dialog using which you can locate the directory in which you would like to save the zip file.

Browse...
 Clicking on this button would open a file dialog using which you can locate the directory in which you would like to save the zip file.

Upload
Click this button to create the zip file and to upload it to the FTP site.

Cancel
Click this button to close the Zip and Upload dialog.
