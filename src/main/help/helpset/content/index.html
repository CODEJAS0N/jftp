Using JFTP Help System

Table of Contents

To view the table of contents, click on the Contents tab in the left frame. The left frame's view will be changed to display the table of contents. Click on a topic to have it displayed. Use the Previous Page and Next Page buttons in the tool bar to navigate within the history of viewed topics.

Printing

To print a help topic, select the topic in the left frame and press the Print button in the tool bar.

To print multiple topics (or all) in one go, select all desired topics in the left frame and click on the Print button. To select multiple topics, hold down the Ctrl ( Command on Mac) key and click on the topic(s).

Please note that selecting a folder in the help topic will not print all the sub-topics underneath it. Instead, you should first expand the folder and select all sub-topics and then press the Print button.
