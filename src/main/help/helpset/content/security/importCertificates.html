Importing Certificate(s)

To import a certificate in to your trusted certificates database, open the certificate manager by selecting Tools -> Manage Certificates... option from the menu bar. This will open the Manage Certificates dialog. Click on the Import button. This will open a file dialog titled, Select a File .

Select the file containing the certificate(s) that you would like to import. Make sure that the file is in DER/PEM encoded format and click on the Open button. This should import the certificate(s) in to your trusted certificate database. If the import fails for any reason, the cause for the failure will be displayed.
