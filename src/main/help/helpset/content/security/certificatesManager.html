Certificate Manager

This is the central point for managing digital certificates. To open the certificates manager, select the menu item Tools -> Manage Certificates... from the menu bar. This will open a new dialog titled Manage Certificates .

This dialog lists all certificates that you have chosen to trust. From this dialog, you can view, import and/or delete certificates.

View
Click on this button to view the details of a certificate. To learn more, follow the instructions given in the section titled, Viewing Details of a Certificate.

This button will be available if you have selected or highlighted one and only one certificate in the certificate list.

Delete
Click on this button to delete one or more certificates from your list of trusted certificates. To learn more, follow the instructions given in the section titled, Deleting Certificate(s).

This button will only be available if you have selected or highlighted one or more certificates in the certificate list.

Import
Click on this button to import certificate(s) from a DER encoded file. To learn more, follow the instructions given in the section titled, Importing Certificate(s).

Close
Click on this button to close the Manage Certificates dialog.
