Installing a Certificate

When connecting to an FTP site using SSL, if the certificate sent by the server is not in your list of trusted certificates, a Security Warning dialog is displayed.

The Security Warning dialog gives you the following information about the certificate.

  • If the date on digital certificate sent by the server is still valid
  • If the name on the digital certificate matches the host name that you are connected to
  • If the certificate was signed by any one of the certificate authorities that you trust

To trust the certificate for this session and continue working with the FTP site, click on the Yes button.

If you do not trust the certificate, click on the No button. This will immediately terminate the connection with the FTP site.

To view and/or install the certificate, click on the View Certificate button. This will open a new dialog titled Certificate Information .

Click on the Install button to add this certificate to your list of trusted certificates. Once a certificate is installed, subsequent connections to that FTP site will be accepted silently as long as the server sends the same certificate.
