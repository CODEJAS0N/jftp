Viewing Details of a Certificate

To view the details of a certificate, open the certificate manager by selecting Tools -> Manage Certificates... option from the menu bar. This will open the Manage Certificates dialog.

In the Manage Certificates dialog, select or highlight the certificate that you are interested in, and click on the View button. This will open the Certificate Information dialog.

The View button will only be available if there is only one certificate selected in the certificate list.
