Editing a Favorite

To edit the connection settings of an existing FTP site in your favorites list, open the favorite manager by doing any of the following:

  • Select Tools -> Manage Favorites menu item from the menu bar
  • Click on the   icon in the tool bar

Doing any of the above actions will open the Manage Favorites dialog.

The Manage Favorites dialog displays a list of all your favorite FTP sites in alphabetical order. From the list, select (highlight) the FTP site that you would like to edit and click on the Edit button. This will open another dialog titled Update Favorite FTP Site dialog.

In the Update Favorite FTP Site dialog, change the connection information as needed and click on the Save button to save your changes. Clicking on the Cancel button would discard your changes and closes the dialog.

For details on various tabs and fields in the Update Favorite FTP Site dialog, please refer to the Adding a Favorite section.
