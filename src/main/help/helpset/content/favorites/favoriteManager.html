Favorite Manager

This is the central point for managing all your favorite FTP sites. To open the favorite manager -

  • Select the menu item Tools -> Manage Favorites... from the menu bar
  • Click on the   icon in the tool bar

Doing any of the above actions will open a new dialog titled Manage Favorites .

This dialog lists all your favorite FTP sites in alphabetical order. From this dialog, you can add, edit and/or delete FTP sites.

Add
Clicking this button opens the Add Favorite FTP Site dialog. Follow the instructions given in the section named Adding a Favorite to add an FTP site to your favorites list.

Edit
Clicking this button opens the Update Favorite FTP Site dialog. Follow the instructions given in the section named Editing a Favorite to update the connection information of an FTP site to your favorites list.

This button will only be available if you have selected (or highlighted) one and only one FTP site in your favorites list.

Delete
Click on this button to delete one or more FTP sites from your favorites list. To learn more, follow the instructions given in the section titled Deleting Favorites.

Please note that this button will only be available if you have selected (or highlighted) at least one FTP site in your favorites list.

Connect
Click on this button to connect to the selected (or highlighted) FTP site in the favorites list. To learn more, follow the instructions given in the section titled Connecting to a Favorite FTP Site.

Please note that this button will only be available if you have selected (or highlighted) one and only one FTP site in your favorites list.

Close
Click on this button to close the Manage Favorites dialog.
