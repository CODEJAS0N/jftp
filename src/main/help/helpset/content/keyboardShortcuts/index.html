Keyboard Shortcuts

The following table lists various keyboard shortcuts that you can use to perform most common tasks.

On Mac OS X, the Ctrl key needs to be substituted with the Command key.

Shortcut Key Action Performed
F1 (Command + ? on Mac OS X) Help
Ctrl + N New Session
F3 Connect...
F4 Disconnect
Ctrl + F3 Reconnect
F7 Download
Ctrl + F7 Download As
Ctrl + Shift + F7 Download and Unzip
F9 Upload
Ctrl + F9 Upload As
Ctrl + Shift + F9 Zip and Upload
F12 Aborts the current transfer immediately.
Ctrl + Shift + O Open local file or directory
Ctrl + Shift + E Edit local file
Ctrl + Shift + P Print local file
Ctrl + Shift + M E-mail local file
Ctrl + Shift + F New Local File
Ctrl + Shift + D New Local Directory
Ctrl + Shift + R Rename local file or directory
Ctrl + Shift + Delete Delete local files and/or directories
Ctrl + Shift + F5 Refreshes the contents of local pane.
Ctrl + Shift + I Local file properties
Ctrl + O Open remote file or directory
Ctrl + E Edit remote file
Ctrl + P Print remote file
Ctrl + M E-mail remote file
Ctrl + F New remote file
Ctrl + D New remote directory
Ctrl + R Rename remote directory
Ctrl + Delete Delete remote files and/or directories
Ctrl + F5 Refreshes the contents of the remote pane.
Ctrl + I Remote file properties
